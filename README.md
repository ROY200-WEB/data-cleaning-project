# data-cleaning-project

## Project Overview

This project focuses on cleaning and preparing raw data using Python and Pandas.

## Tasks Performed

* Identified missing values
* Removed duplicate records
* Corrected invalid data entries
* Standardized date formats
* Validated cleaned dataset

## Technologies Used

* Python
* Pandas
* Jupyter Notebook

## Output

A cleaned dataset exported as `cleaned_dataset.csv`.

## Skills Gained

* Data Cleaning
* Data Preparation
* Handling Missing Values
* Removing Duplicates
* Data Formatting
